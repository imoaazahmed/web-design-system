# Category Listing Page

## Installation

Design System can be added as a dependency from
[here](https://github.com/tradeling/web-design-system-sdk/releases).

## Import

```
import { ReadyToShipProduct, SupplierCard } from '@tradeling/web-design-system';
```

## ReadyToShipProduct Usage

```
<ReadyToShipProduct>
    <ReadyToShipProduct.Image src="https://cdn.britannica.com/68/143268-050-917048EA/Beef-loin.jpg" alt="Shredded Chicken with Salad" />
    <ReadyToShipProduct.Content>
        <ReadyToShipProduct.Name>Shredded Chicken with Salad</ReadyToShipProduct.Name>
        <ReadyToShipProduct.Price
            minPrice={26.0}
            maxPrice={96.0}
            currency="AED"
        />
        <ReadyToShipProduct.Units value={19} />
        <ReadyToShipProduct.Supplier>
            Hangzhou Aiyomi Food Co., Ltd.
        </ReadyToShipProduct.Supplier>
        <ReadyToShipProduct.ExtraInfo />
        <Button isFullWidth variant="outline">
            Contact Supplier
        </Button>
    </ReadyToShipProduct.Content>
</ReadyToShipProduct>
```
